#### バグ修正（YAML インポート）
人間による動作確認で Articulation が読み込まれないバグを報告された。

**原因**: YamlDotNet はシーケンスを親マッピングキーと同じインデントで出力するが（`Articulations:\n- Name: ...`）、パーサーが「子は必ず親より深いインデント」と判定していたため読み込みをスキップしていた。また UTF-8 BOM が付与されたファイルでパースが失敗する問題もあった。

**修正内容**: 同インデントの `- ` 行も子シーケンスとして受け入れるよう条件を変更。パース前に BOM（`﻿`）を除去する処理を追加。
