# エディタのデザイン改善

## 課題

現状のデザインはほぼ白一色で、エディタのUIが視認しづらい。エディタのUIをマテリアルデザインを採用したカラーでデザインすることで、エディタの視認性を向上させたい。

## ゴールの詳細

エディタの視認性が向上し、ユーザビリティが向上すること。

### 成果物の要件

- マテリアルデザインを採用したカラーで各エディタのUIがデザインされていること
- ヘッダ背景色
- ボタン背景色・テキスト色
- 各種タイトル領域の背景色・テキスト色
